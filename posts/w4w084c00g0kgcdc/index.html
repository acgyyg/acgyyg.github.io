作品内容

故事梗概

某种原因导致阿波罗空间站陷入混乱之中。作为实验对象的外星人，现在是展现其魔女身份的时刻。只需履行义务，逃离这个地方即可。

游戏内容

・符文魔法打倒敌人并获取灵魂碎片，学习魔法。通过4种不同的符文组合，可实现20种以上的魔法。每种魔法都有独特的用途。可以同时组合最多3个符文，触发所需的魔法。自己创造独特的游戏风格吧。

・阿波罗空间站阿波罗空间站直到某个事件发生前都是一个普通的站点。这混乱的情况并非偶然。或许幕后隐藏着一些秘密…

・难度设定难度分为简单、普通、困难三种。大多数敌人会根据难度展现不同的行动。难度增加会增加敌人的攻击模式。改变难度进行游戏，会带来全新的体验。

・束缚与逃脱大多数敌人会根据女主角的暴露程度而兴奋。敌人兴奋时会试图使她昏迷并抓住她。女主角可以尝试抵抗。如果失败，女主角需要从俘虏中逃脱。然后，她需要回到掉落魔杖的地方将其拾起。此外，有些敌人会在女主角昏迷后就在原地对她为所欲为。

※有些敌人无法抵抗。※也有些情况下无法逃脱。 – 特殊输入键 –特殊输入键可以在控制角色时随时输入。 ・Ctrl + C + H无敌模式※但游戏仍可能因无法逃脱而失败。 ・Ctrl + C + M无限法力模式 ・Ctrl + C + G获得9999个灵魂碎片如果想立即使用所有魔法，请自由使用此命令。 ・Ctrl + C + O无H模式女主角的服装不会被破坏。不会发生H事件。 若要关闭，请再次输入相同的键。

游戏系统

  1. 现在的HP。在无敌模式下会变成红色。

  2. 现在的MP。在无限魔法模式下会变成红色。

  3. 现在的体力。仅在抵抗敌人捕获时使用。只有在使用保存点时才会恢复。

  4. 现在的精神状态。每次H事件发生时都会变红。如果完全变成红色，则无法在H事件后逃脱，游戏会立即结束。在没有H事件的模式下，核心不会显示。只有在使用保存点时才会恢复。

  5. 因精神状态下降导致的最大HP减少。黑色部分表示减少量。只有在使用保存点时才会恢复。

– 选项 –· 操作方法您可以在选项中确认操作方法。左键通常是攻击，右键除了魔法咏唱外，可以重新绑定为任意按键。

· 性能模式通过降低帧率来提高性能。这是使游戏运行更流畅的最后手段。只有在您真正感到需要时才应打开。

自购证明

Noraheart名义上的第二作，也是我认为目前在Noraheart的四个作品中游戏性和实用性最高的一作，同时也是这作开始，Noraheart的人体终于正常了😭，虽然动作可能还是有点僵硬

Noraheart在这作对游戏玩法创新挺不错，4种不同的符文，可以组合成20多种不同的魔法，每个魔法的效果都不一样，有的是攻击型的、有的是辅助型。整体体验下来，这个符文系统还是很有意思的。束缚与逃脱玩法，有点类似与被带回巢穴，但这个玩法有一点比较史，就是在比较远的地方被抓了，还得跑一大段路才能拿回法杖☹️

实用性方面，这作对比《SusRitual》可谓是有了极大的提升，首当其冲的就是那糟糕的人体在这作终于正常了，CG也全都是动态的，画风也是开始逐渐定型了，我是非常喜欢😋

总的来说，如果bro们喜欢玩ACT的，我个人挺推荐的。

当然，具体好不好玩还是得看看bro们的评价。
