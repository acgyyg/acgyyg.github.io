简介

《异常咖啡机 2》 是一款奇特的互动视觉小说与虚构模拟游戏。 你站在一位神秘少女和一台不起眼的咖啡自动售卖机前，这台机器能生产出你输入的任何东西——无论看起来多么不可能。 她不会告诉你自己是谁、为什么会在那里、或者这台咖啡机的来历。 她唯一的要求是：输入一个词——任何词——然后按下按钮。 机器会回应。它总是会回应。 输入 “coffee”，你会得到一杯咖啡。 输入 “blood”——是的，它也会给你。 输入 “saliva”——你大概不该这么做，但它也能做到。 输入 “death”？你可以试试。

玩法说明 玩法很简单：

输入一个词。

一只杯子出现。

选择谁来喝：少女，还是你自己。

接着，会发生一些事——或者什么都不发生——又或者，需要等待。

她会根据你给她的东西作出不同反应。 有时她会微笑。 有时她会皱眉。 有时她会消失，随后以不同的样子回来。 有时，她再也不会回来。 如果你选择自己喝下去，那么机器就会对你作出反应。 别指望是等价交换。 没有说明书。 没有任务。 没有路线标记。 没人告诉你这台机器是什么、想要什么。 你只能自己去实验。 尝试各种单词。 试探各种情绪。 推动它。 破坏它。 或者让它毁掉你。 这不是一款关于“胜利”的游戏。 甚至不是关于“理解”的游戏。 它关乎好奇心。 关乎你愿意走多远。 关乎你选择给她什么、给自己什么，以及之后会发生什么。 没有手册。 没有结局。 只有少女、机器，以及你接下来要输入的那个词。 所以——去吧。 试一个词。 然后，喝下去。 你能找到游戏里所有的关键词吗？
